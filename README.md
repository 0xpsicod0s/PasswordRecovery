# Password Recovery

Software desenvolvido para recuperação de senhas

---
## Instalação

```
$ git clone https://github.com/cxnd3v/PasswordRecovery.git

$ cd PasswordRecovery

$ pip install -r requirements.txt

$ python ./login_aes.py
```
